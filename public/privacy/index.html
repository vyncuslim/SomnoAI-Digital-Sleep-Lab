Privacy Policy

Last Updated: January 4, 2026

1. Data Access

SomnoAI accesses Google user data only after explicit user consent. The data accessed may include:

  • Sleep Segments
  • Heart Rate
  • Activity Energy Expenditure

2. Data Usage

The accessed data is used solely for:

  • Personal sleep visualization
  • AI-based sleep pattern analysis
  • Generating individualized insights within the application

SomnoAI does not use Google user data for advertising, marketing, resale, or AI model training.

3. Data Storage

SomnoAI operates with a zero-server-storage architecture.

  • No user data is stored on external servers
  • All processing occurs locally in the browser session
  • Data is stored temporarily in sessionStorage
  • Data is permanently deleted upon logout or tab closure

4. Data Sharing

SomnoAI does not share Google user data with third parties.

5. User Control & Rights

Users may revoke SomnoAI’s access at any time via their Google Account settings.

6. Google API Limited Use Disclosure

SomnoAI’s use of information received from Google APIs adheres to the Google API Services User Data Policy , including the Limited Use requirements.
