隐私权政策 (Privacy Policy)

最后更新日期： 2024年3月25日

欢迎使用 SomnoAI Digital Sleep Lab。我们非常重视您的隐私。本政策旨在说明我们如何处理您的个人健康数据。本应用是一个纯前端的数字实验室，专注于保护您的生理隐私。

Google API 有限使用披露 (Limited Use Disclosure)

SomnoAI 对从 Google API 接收的信息的使用和传输将遵守 Google API 服务用户数据政策，包括其中的“有限使用”要求。您的数据绝不会被用于广告投放、个人画像构建或出售给第三方机构。

1. 数据访问与收集

当您选择“接入 Google Fit”时，本应用会请求访问以下数据：

  • 睡眠分段数据： 用于分析您的睡眠阶段（深睡、REM、浅睡、清醒）。
  • 心率数据： 用于计算静息心率及其在睡眠期间的变化趋势。
  • 活动消耗数据： 用于评估全天能耗对睡眠质量的影响。

这些数据的获取是基于您的显式授权，且仅用于为您生成数字化的睡眠实验报告。

2. 数据处理与存储声明

零服务器架构 (Zero Backend)： 这是 SomnoAI 的核心隐私承诺。本应用不设后端服务器，不具备云端持久化存储能力。所有数据流均在您的浏览器本地内存中实时处理。

  • 会话存储： 数据仅保存在浏览器的 sessionStorage 中。
  • 自动抹除： 一旦您关闭浏览器标签页或点击“退出登录”，所有从 Google Fit 同步的数据将立即从您的设备中永久消失，无法找回。

3. 数据的共享与披露

我们不会与任何第三方共享、出售、出租或交易您的健康数据。您的所有生理信号仅供您本人在当前会话中查阅。

4. AI 洞察服务

应用内集成 Gemini AI 引擎为您提供睡眠分析。在此过程中，系统仅将必要的脱敏指标（如：深睡比例、静息心率数值）发送给 AI 模型以生成建议，不会附带任何身份识别信息（如姓名、邮件地址）。

5. 撤回授权与权利

您可以随时通过以下方式停止数据收集：

  • 在应用设置中点击“退出并重载”。
  • 在您的 Google 账号设置中的“安全性”选项下，撤销对 SomnoAI 的授权。
