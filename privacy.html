Google API 数据访问说明

当您授权连接 Google Fit，本应用会访问您的 睡眠分段 (sleep segments)、心率 (heart rate) 及 活动能量 (activity energy) 数据，仅用于数字睡眠分析。

数据使用 (Data Usage)

所有数据仅用于应用内可视化与 AI 分析，不会用于广告投放、个人画像或任何形式的第三方分享。Somno 对从 Google API 接收的信息的使用将遵守 Google API 服务用户数据政策，包括其中的“有限使用”要求。

数据存储 (Data Storage)

数据只存储在浏览器本地 sessionStorage 中。本应用不设立任何后端数据库，一旦关闭页面或手动登出，所有数据将立即从内存中彻底删除。

用户权利 (Your Rights)

您可随时在 Google 账号设置中的“第三方应用与服务”部分撤销对本应用的授权。
