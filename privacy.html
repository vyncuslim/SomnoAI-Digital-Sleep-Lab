隐私权政策 (Privacy Policy)

生效日期： 2024年3月25日

欢迎访问 sleepsomno.com。我们通过零后端、纯前端的数字化架构保护您的生理隐私。本政策说明我们如何处理您的 Google Fit 健康数据。

Google API 有限使用披露 (Limited Use Disclosure)

SomnoAI 对从 Google API 接收的信息的使用和传输将遵守 Google API 服务用户数据政策，包括其中的“有限使用”要求。我们绝不将您的健康信息用于广告投放、个人画像构建或出售给任何第三方。

1. 数据访问与用途

仅在您显式点击“授权接入”后，本应用才会通过 Google OAuth 访问以下指标：

  • 睡眠分段： 用于识别和可视化您的睡眠周期。
  • 心率数据： 用于计算静息心率及其睡眠期间的波动。
  • 活动消耗： 用于分析日常代谢对睡眠深度的影响。

2. 零存储架构 (Zero Storage)

数据仅保存在您本地浏览器的 sessionStorage 中。本应用没有后端服务器，这意味着一旦您关闭标签页，所有同步的数据将即刻从设备中永久抹除。

3. 撤销权限

您可以随时访问 Google 安全设置 撤回对 SomnoAI 的授权。
