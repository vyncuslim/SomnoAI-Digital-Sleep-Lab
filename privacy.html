Somno Digital Sleep Lab (sleepsomno.com) 承诺保护您的个人隐私。本协议详细披露我们如何通过 Google API 访问您的健康数据，以及我们如何利用这些生理信号提供科研级的睡眠分析。

Google API 有限使用披露 (Limited Use Disclosure)

Somno 对从 Google API 接收的信息的使用和向任何其他应用的转移，将遵守 Google API 服务用户数据政策，包括其中的“有限使用”要求。

隐私承诺： 我们绝不将您的 Google Fit 数据用于广告投放、个人画像或任何形式的第三方商业交易。您的所有健康指标仅用于生成本应用内的睡眠报告。

1. 数据访问范围 (Data Accessed)

本实验室仅请求访问以下数据集的只读权限：

  • SCOPE_01
    睡眠架构 (com.google.sleep.segment)

    用于提取深度睡眠、REM、浅睡及清醒阶段。这些数据是 AI 合成睡眠评分的核心。

  • SCOPE_02
    生理频率 (com.google.heart_rate.bpm)

    用于读取静息心率及其波动。我们利用此数据评估您的自主神经系统恢复状态。

  • SCOPE_03
    能量代谢 (com.google.calories.expended)

    用于将全天能量消耗与睡眠效率进行交叉对比，为您提供更精准的恢复建议。

2. 处理逻辑与用途 (Usage)

访问的数据通过以下流程进行处理：

  • 睡眠架构深度分析： 分析不同睡眠阶段的连续性，合成您的每日数字化睡眠得分。
  • 生理恢复投影： 利用 AI 神经引擎预测次日的认知负荷、专注力水平及体能恢复程度。
  • 个性化建议： 基于脱敏后的指标，为您提供包括睡眠环境优化、营养补充（如镁）等在内的科学建议。

3. 零后端与边缘计算架构 (Zero-Backend)

本实验室采用 隐私优先的分布式设计，确保您的数据绝对主权：

无后端数据库

Somno 绝不在云端存储您的生理指标。数据仅存在于您的设备前端。

会话级隔离

所有健康数据流仅暂存于 sessionStorage。页面关闭或手动登出，所有数据即刻物理销毁。
