隐私权政策 / Privacy Policy

最后更新日期：2024年3月25日

SomnoAI Digital Sleep Lab（以下简称“我们”或“本应用”）非常重视您的隐私。本政策旨在说明我们如何处理通过 Google Fit 访问的健康数据。

SomnoAI Digital Sleep Lab ("we", "us", or "the App") values your privacy. This policy explains how we handle health data accessed via Google Fit.

Google API 有限使用披露 / Limited Use Disclosure

EN: SomnoAI's use and transfer to any other app of information received from Google APIs will adhere to the Google API Service User Data Policy, including the Limited Use requirements.

ZH: SomnoAI 对从 Google API 接收到的信息的使用和传输将严格遵守 Google API 服务用户数据政策，包括其中的“有限使用”要求。

1. 数据收集范围 / Data Collection

当您授权连接 Google Fit 时，我们会访问以下数据：

  • 睡眠数据 (Sleep Data): 睡眠时间、睡眠阶段（深睡、REM、浅睡）。
  • 生理指标 (Vital Signs): 心率 (Heart Rate)。
  • 活动数据 (Activity Data): 每日卡路里消耗量。

2. 数据用途 / Data Usage

我们收集的数据仅用于生成可视化的睡眠质量图表和通过 AI 提供健康建议。我们绝不将您的健康数据用于广告或任何形式的第三方营销。

3. 数据存储与安全 / Storage & Security

端侧处理声明：本应用不设后端服务器。所有数据仅存储在您浏览器本地的 Session Storage 中，一旦关闭标签页，数据将立即被抹除。我们不具备在云端持久化存储您个人健康数据的技术架构。
