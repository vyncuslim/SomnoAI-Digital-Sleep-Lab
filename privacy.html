Google API Limited Use Disclosure

SomnoAI's use and transfer to any other app of information received from Google APIs will adhere to the Google API Service User Data Policy, including the Limited Use requirements.

1. 数据访问与授权 / Data Access

当您连接 Google Fit 时，本应用会申请访问您的睡眠数据（Sleep）、心率（Heart Rate）和活动消耗（Activity）。

We access your health data from Google Fit only with your explicit consent.

2. 数据用途 / Data Usage

数据仅用于生成您的睡眠质量图表和提供 AI 建议。我们绝不将您的健康信息用于广告投放或出售给第三方。

Data is used solely for personalized analysis and is never used for advertising.

3. 数据存储 / Data Storage

零服务器存储声明：本应用不设后端服务器。所有数据仅存储在您浏览器的 sessionStorage 中，且仅在当前会话有效。一旦关闭标签页，数据将被永久删除。

No health data is stored on any cloud server. All processing is local to your browser.
