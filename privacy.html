隐私权政策 / Privacy Policy

最后更新日期：2024年3月25日 | Last Updated: March 25, 2024

SomnoAI Digital Sleep Lab（以下简称“我们”或“本应用”）尊重并保护您的隐私。本应用通过访问 Google Fit 的健康数据为您提供睡眠分析服务。

SomnoAI Digital Sleep Lab ("we", "us", or "the App") respects your privacy. We provide sleep analysis by accessing Google Fit data.

Google API 有限使用披露 / Limited Use Disclosure

EN: SomnoAI's use and transfer to any other app of information received from Google APIs will adhere to the Google API Service User Data Policy, including the Limited Use requirements.

ZH: SomnoAI 对从 Google API 接收到的信息的使用和传输将严格遵守 Google API 服务用户数据政策，包括其中的“有限使用”要求。

1. 数据访问与收集 / Data Access

我们仅在获得您明确授权后访问以下 Google Fit 数据：

  • 睡眠段 (Sleep Segments): 用于分析您的睡眠时长和阶段。
  • 心率 (Heart Rate): 用于推算静息心率和健康趋势。
  • 活动消耗 (Calories): 辅助评估全天代谢水平。

2. 数据用途 / Data Usage

您的健康数据仅用于在应用内生成可视化图表和 AI 建议。我们绝不会将这些数据出售给第三方，也绝不会将其用于广告定位。

3. 数据存储 / Data Storage

零服务器存储 (Zero Server Storage): 本应用没有后端服务器。所有获取的数据仅存储在您浏览器本地的 sessionStorage 中，且仅在当前会话有效。关闭浏览器标签页或退出登录后，数据将立即被永久删除。
