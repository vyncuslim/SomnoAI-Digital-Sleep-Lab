隐私权政策 (Privacy Policy)

最后更新日期：2024年3月24日

1. 简介 (Introduction)

SomnoAI Digital Sleep Lab（“本应用”）是一款旨在帮助您了解睡眠模式的数字化工具。本隐私政策解释了我们如何处理通过 Google Fit API 访问的数据。

Google API 有限使用披露 (Limited Use Disclosure)

SomnoAI 对从 Google API 接收到的信息的使用和传输将遵守 Google API 服务用户数据政策，包括其中的“有限使用”要求。

2. 数据收集与使用 (Data Collection & Use)

我们仅在您通过 Google OAuth 授权屏幕明确许可后，访问以下数据类别：

  • 睡眠阶段：用于可视化您的深睡、REM、浅睡及清醒时间。
  • 心率指标：用于计算静息心率并分析身体恢复情况。
  • 活动与能耗：用于关联体力活动与睡眠质量的关系。

我们绝不向第三方出售您的数据，不将数据用于广告目的，也不使用您的健康数据训练 AI 模型。

3. 数据安全 (Data Security)

所有数据处理均在您的浏览器本地进行。我们不设后端服务器，所有健康数据存储在浏览器的 Session Storage 中，关闭标签页即刻清除。
