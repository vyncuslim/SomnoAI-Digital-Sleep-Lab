Google API Limited Use Disclosure

SomnoAI's use and transfer to any other app of information received from Google APIs will adhere to the Google API Service User Data Policy, including the Limited Use requirements.

1. 数据访问与收集 / Data Collection

本应用仅在您的浏览器本地处理 Google Fit 的睡眠与生理数据。我们不会通过任何网络请求将您的个人健康数据传输至我们的服务器。

2. 数据存储 / Data Storage

所有数据仅存储在浏览器会话（sessionStorage）中。关闭浏览器标签页后，所有数据将立即被抹除。
