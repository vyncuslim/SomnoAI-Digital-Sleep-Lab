隐私权政策 (Privacy Policy)

生效日期：2024年3月20日 | 最近更新：2024年3月22日

1. 引言

SomnoAI Digital Sleep Lab（以下简称“本应用”）非常重视您的隐私。本政策旨在说明我们如何收集、使用、存储及保护您的健康数据。本应用是一个纯前端驱动的实验性工具，旨在通过可视化与 AI 洞察提升您的睡眠健康意识。

2. 数据收集范围与目的

我们仅在您通过 Google OAuth 授权后，访问以下特定范围的健康数据：

  • 睡眠分段 (Sleep segments)：用于重构您的睡眠架构图谱（如深睡、REM、浅睡）。
  • 心率数据 (Heart rate)：用于分析睡眠期间的静息心率趋势。
  • 能量消耗 (Calories expended)：用于评估日间代谢与睡眠质量的关联。

收集上述数据的唯一目的是在您的浏览器端生成个人健康洞察和可视化报告。

3. Google API 有限使用政策合规声明

SomnoAI 对从 Google API 接收到的信息的使用和传输将遵守 Google API 服务用户数据政策，包括其“有限使用”要求。我们不会将您的 Google Fit 数据用于广告、用户分析或任何与增强本应用核心功能无关的目的。

4. 数据存储与共享

零服务器存储：本应用不设任何后端服务器。所有从 Google Fit 提取的数据仅在您浏览器的 SessionStorage 中进行临时缓存，一旦您关闭标签页或点击“退出”，数据即被销毁。

禁止第三方共享：我们绝不会向任何第三方（包括广告商、数据经纪人或保险商）出售、租赁或披露您的健康数据。您的数据永远保留在您的设备本地。

5. AI 处理说明

本应用使用 Google Gemini API 生成睡眠建议。在此过程中，我们会将匿名化的指标（如睡眠分数、心率数值）发送至 Gemini 引擎进行逻辑推演。我们不会在发送给 AI 的 Prompt 中包含您的个人身份信息（如姓名或电子邮箱）。

6. 您的权利与选择

您可以随时：

  • 拒绝授权或在授权时取消勾选特定的数据类型。
  • 在应用设置中点击“退出并重载”以清除所有本地缓存。
  • 在您的 Google 账号设置 中撤销对 SomnoAI 的全部访问权限。

7. 政策变更

我们可能会不时更新本隐私权政策。任何变更都将通过在本页面发布新政策并更新“生效日期”的方式通知您。建议您定期查看。
