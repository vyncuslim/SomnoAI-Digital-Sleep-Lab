隐私权政策

合规更新日期： 2024年5月22日

Google API 有限使用披露

本应用严格遵守 Google API 服务用户数据政策。健康数据绝不用于商业广告或第三方交易。

1. 数据访问说明 (Data Accessed)

在授权下，我们访问：Google Fit 睡眠阶段数据（读取睡眠架构）、心率生理指标（读取恢复水平）及能量消耗数据。

2. 数据使用用途 (Data Usage)

数据仅用于本地生成睡眠分数、AI 深度洞察建议及可视化健康趋势图。所有分析逻辑均在客户端边缘端运行。

3. 存储与共享规范 (Storage & Sharing)

数据不经后端服务器存储，仅暂存在浏览器本地 session 缓存中，且绝不与第三方共享。页面关闭即刻销毁。
