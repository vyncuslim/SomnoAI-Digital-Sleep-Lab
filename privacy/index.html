Google API Limited Use Disclosure

SomnoAI's use and transfer to any other app of information received from Google APIs will adhere to the Google API Service User Data Policy, including the Limited Use requirements.

1. 数据收集 / Data Collection

我们仅在获得您的明确授权后，从 Google Fit 访问您的睡眠段数据、心率和卡路里消耗。所有数据均仅用于应用内的分析展示。

We access sleep segments, heart rate, and calories from Google Fit only with your consent.

2. 数据存储 / Data Storage

零服务器存储：本应用不设后端服务器。所有数据仅存储在您浏览器的本地会话 (sessionStorage) 中。关闭标签页或退出登录将立即清除所有本地缓存。

We have no backend servers. All data is stored locally in your browser's session storage and is wiped upon session end.
